# Multi-Agent AI Research Assistant Frontend

This React + TypeScript + Tailwind frontend provides a modern dashboard for the AI research assistant.

## Setup

1. Copy `.env.example` to `.env`
2. Install dependencies:

```bash
cd frontend
npm install
```

3. Run the app:

```bash
npm run dev
```

4. Open the site at `http://localhost:5173`

## Features

- Research workflow UI
- PDF upload interface
- Document chat page
- Dark mode toggle
- API integration with backend service
